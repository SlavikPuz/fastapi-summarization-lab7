При выполнении лабораторной работы возникла проблема с использованием изначально выбранной модели: её вес был слишком большим, из-за чего загрузка постоянно прерывалась и не удавалась ни одним из доступных способов. После двух часов страданий, было принято решение заменить модель на более лёгкий вариант. Смысл модели абсолютно тот же, а именно - суммаризация текста.
