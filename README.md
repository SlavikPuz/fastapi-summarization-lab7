При выполнении лабораторной работы возникла проблема с использованием изначально выбранной модели: её вес был слишком большим, а скорость очень маленькой(почти нулевой), из-за чего загрузка постоянно прерывалась и не удавалась ни одним из доступных способов. После двух часов страданий, было принято решение заменить модель на другую, менее тяжелую. Смысл модели абсолютно тот же, а именно - суммаризация текста.
